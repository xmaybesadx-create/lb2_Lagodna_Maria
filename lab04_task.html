Початок роботи з HTML

Що таке HTML?

HTML (HyperText Markup Language – мова гіпертекстової розмітки) не є мовою програмування – це мова розмітки, яка використовується для визначення структури веб-сторінок, які відвідують користувачі. Вони можуть мати складну чи просту структуру, все залежить від задуму та бажання веб-розробника. HTML складається з ряду елементів, які ви використовуєте для того, щоб охопити, обернути або розмітити різні частини вмісту, щоб він мав певний вигляд або спрацьовував певним способом.

Вбудовані теги можуть перетворити частину вмісту на гіперпосилання, за якою можна перейти на іншу веб-сторінку, виділити курсивом слова і так далі. Наприклад:

My cat is very grumpy

Як абзац:

<p>My cat is very grumpy</p>

Примітка. Теги HTML нечутливі до регістру... Найкращою практикою є запис у нижньому регістрі.

Структура HTML-елементів

Давайте розглянемо елемент "параграф" трохи докладніше:

Основними частинами елемента є:

  • Відкриваючий тег
  • Закриваючий тег
  • Вміст
  • Елемент

Вкладені елементи

Ви також можете вкладати елементи всередину інших елементів – це називається вкладеністю. Якщо ми хочемо підкреслити, що наш кіт дуже сердитий, ми можемо укласти слово "very" в елемент <strong>, який означає, що це слово вкрай важливо в даному контексті:

<p>My cat is <strong>very</strong> grumpy.</p>

Ви повинні переконатися, що елементи вкладені належним чином: у наступному прикладі відкриваємо p елемент першим, потім елемент strong, потім ми закриваємо елемент strong першим, потім p. Нижче наведено приклад неправильного вкладення:

<p>My cat is <strong>very grumpy.</p></strong>

Елементи мають відкриватися та закриватися правильно, таким чином, щоб явно перебувати всередині чи зовні один одного. Якщо вони перекриваються так, як у прикладі вище, ваш браузер спробує «додумати» за вас, що ви мали на увазі, і ви отримаєте непередбачуваний результат. Тож не робіть так!

Порожні (void) елементи

Не всі елементи відповідають вищезгаданому шаблону: відкриваючий тег, контент, закриваючий тег. Деякі елементи складаються з одного тегу і зазвичай використовуються для вставки чогось у місце документа, де розміщені. Наприклад, елемент <img> вставляє картинку на сторінку в тому самому місці, де він розташований:

<img src="https://drive.google.com/file/d/1a7ZPfTHNyJBOMHR5WsF9Rg5sBpME8gKm/view?usp=sharing">

Примітка. У HTML можна (але не обов’язково) закривати пусті теги: <img ... />

Атрибути

Атрибути містять додаткову інформацію про елемент...

  • href – веб-адреса посилання
  • title – текст підказки
  • target – де відкривається посилання

Бульові атрибути

<input type="text" disabled="disabled">
<input type="text" disabled>
<input type="text">

Опускання лапок

<a href=https://www.mozilla.org/>favorite website</a>
<a href=https://www.mozilla.org/ title=The Mozilla homepage>favorite website</a>

Порада: завжди використовуйте лапки для атрибутів.

Структура HTML документа

<!DOCTYPE html>
<html lang="en-US">
  <head>
    <meta charset="utf-8" />
    <title>My test page</title>
  </head>
  <body>
    <p>This is my page</p>
  </body>
</html>

Пояснення: <!DOCTYPE html> ... <body> – містить увесь видимий вміст.

Пробіли в HTML

<p>Cats are silly.</p>
<p>Cats        are
         silly.</p>

Браузер скорочує кілька пробілів до одного. Форматування робить код читабельним.

Спеціальні символи HTML (Entity)

Наприклад:

<p>In HTML, you define a paragraph using the <p> element.</p>

Примітка. Для <, >, & використовуйте коди: &lt;, &gt;, &amp; тощо.
